principal CSS Regiões

Se você não vir dois blocos de texto lado a lado, as regiões CSS não são suportadas.

A exibição de layouts complexos de uma revista, jornal ou livro didático típico na Web requer recursos além daqueles disponíveis nos módulos CSS existentes. O layout dinâmico da revista, em particular, requer flexibilidade na colocação de caixas para fluxos de conteúdo. Este é o propósito do módulo de regiões CSS.

Os módulos de regiões CSS permite que o conteúdo flua por várias áreas chamadas regiões. As regiões não são necessariamente contíguas na ordem do documento. O módulo de regiões CSS fornece um mecanismo avançado de fluxo de conteúdo, que pode ser combinado com esquemas de posicionamento definidos por outros módulos CSS, como o Módulo Multi-Coluna ou o Módulo de Layout de Grade para posicionar as regiões onde o conteúdo flui.
